# Pymodoro

A simple Pomodoro timer that lives in your tray and does nothing until you set a timer.

![Pymodoro screenshot](./assets/screenshot.png)
![Pymodoro screenshot 2](./assets/screenshot2.png)

- Simple GUI via TKinter
- screeninfo 0.8.1
- pystray 0.19.5
- Pillow 10.3
- playsound 1.3.0
- Python 3.12.3
